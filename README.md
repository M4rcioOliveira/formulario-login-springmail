#  Sobre

Esse projeto consiste na criação de uma formulário de cadastro de login utilizando HTML, CSS e  BOOTSTRAP no front, e Java + SpringMail no backend para confirmação de cadastro.
## Tecnologias
 -  Java
 - SpringBoot
 - SpringMail
 - Maven
 - Jquery
 - HTML
 - CSS
 - BootStrap

## Features

 - [ x ] Validação de Cadastro via email
 - [ x ]  Confirmação de Cadastro via email

## Passos para Executar o Projeto

 - Clone esse projeto para sua máquina
 - Configure os parâmetros email e password na pasta resource --> arquivo properties na pasta raiz do projeto.
 - Execute a API
 -   Crie um banco de dados. Obs: Banco utilizado -> postgree.
 - - Caso utilize outro, configura as os parâmetros de conexão na pasta resource --> arquivo properties  na pasta raiz do projeto.
 -  Abra o arquivo index em um navegador
 
 - Tutoriais úteis: https://receitasdecodigo.com.br/spring-boot/como-configurar-projetos-spring-boot-para-enviar-e-mail
